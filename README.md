# GoodooPay iOS SDK

Public iOS binary SDK mirror for GoodooPay.

## Contents

- `ios/goodoo_pay.podspec`
- `ios/Classes/` - native iOS bridge API
- `ios/Frameworks/` - vendored Flutter/App/plugins XCFrameworks
- `example/` - runnable native iOS example

## Validate Podspec

```bash
pod lib lint ios/goodoo_pay.podspec --allow-warnings
```

## Versioning and tags

- SDK version: `1.0.0`
- Tag format: `ios-vX.Y.Z`
